Position property has 4 Options

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

static

static is the default value for the position proeprty

top, bottom, left, and right will NOT work with static

this element will show up, top to bottom, left to right, nothing speical and it cannot be moved

Relative

a relative position is the same as static(top to bottom, left to right), EXCEPT you can move it with top, bottom, left and right

if you move it, it will sit on top of other stuff and the rest of the page will think it's still where "static" would be

if you don't give it anything, it will behave like a static

fixed

a fixed position element will remain relative to the viewport(screen). Even if the user scrolls, the screen doesnt move, so the element doesnt move either

unlike relative elements, it does not leave a gap, the rest of the page(static, relative things) don't know it exists anymore

Absolute

absoluate will stick to it's parent, until it reaches another position thats NOT static, what we can do is add a relative to it's parents, so that it's an anchor
